# ◇ 抽乌帕！

{% hint style="info" %}
抽乌帕中的功能<mark style="color:green;">群聊私聊均可使用</mark>，但是放下乌帕将会<mark style="color:green;">收取2乌帕（为数不多可以获取乌帕的渠道）</mark>
{% endhint %}

输入以下内容，根据提示进行即可

> /抽乌帕

查看拥有的奖券

> /奖券列表

{% hint style="info" %}
注意：

* 根据放下乌帕数量的不同对于抽出的结果有增益或者减益
* 你所放下的乌帕都可能被另外一个人捡起
* 抽乌帕将会触发随机事件（~~<mark style="color:green;">包括了亿点点私货和某只客串的蠢萝莉</mark>~~）
{% endhint %}
